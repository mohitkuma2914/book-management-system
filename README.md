# 📚 Book Management System

Book Management System is a CRUD REST API built using Spring Boot. 
It allows users to add, view, update, and delete book records. 
The project demonstrates backend development using Spring Boot, REST APIs, 
and database integration for managing book information.

---

## 🚀 Features
- Add a new book
- View all books
- Update book details
- Delete a book
- RESTful API design

---

## 🛠 Technologies Used
- Java
- Spring Boot
- MySQL
- REST API
- Maven

---

## 📂 Project Structure
src/main/java
 ├── controller
 ├── service
 ├── repository
 └── model

---

## 🔗 API Endpoints

| Method | Endpoint | Description |
|------|------|------|
| POST | /books | Add a new book |
| GET | /books | Get all books |
| PUT | /books/{id} | Update book |
| DELETE | /books/{id} | Delete book |

---

## ⚙️ How to Run the Project

1. Clone the repository
2. Open the project in IntelliJ / VS Code
3. Configure MySQL database in `application.properties`
4. Run the Spring Boot application
